# questioner-api

simple dynamic Q/A api
___
### dependencies
* MongoDB v3.4.12
* NodeJS v8.12.0
___
### node packages
* body-parser v1.18.3
* express v4.16.4
* mongoose v5.3.8
___
### basic dev/prod
1. `npm setup` 
  will migrate questions from `./app/storage/qustions.json` to questions collection
2. `npm start`
  will start the server on `localhost:3000`
